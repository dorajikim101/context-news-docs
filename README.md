# 컨텍스트 뉴스 개발 문서 인덱스

이 폴더는 **컨텍스트 뉴스 1차 MVP 개발 착수용 문서 세트**를 담는다.

## 읽는 순서
1. `PRODUCT-MVP-SCOPE.md`
2. `TECH-SPEC.md`
3. `UX-OPS-SPEC.md`
4. `P0-SPRINT-PLAN.md`

## 문서 목적
- **PRODUCT-MVP-SCOPE**: 왜 만들고, 이번 1차에서 무엇을 만들고 무엇을 안 만드는지
- **TECH-SPEC**: 실제 구현에 필요한 기술 구조와 DB/API/배포 기준
- **UX-OPS-SPEC**: 내부 편집 흐름, 외부 읽기 화면, 운영 방식
- **P0-SPRINT-PLAN**: 바로 개발 시작할 때 첫 우선순위와 완료 기준

## 현재 기준
- 도메인: 크립토
- MVP 본체: 반자동 서사 편집 CMS + 읽기 화면
- 핵심 목표: narrative / claim / evidence / counterpoint / state 구조화 검증

## 참고
기존 v0.x 문서들은 아이디어와 설계 변천 기록이며, 실제 착수는 위 4개 문서를 우선 기준으로 한다.
