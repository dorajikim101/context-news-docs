# Context News Frontend

Next.js frontend scaffold for Context News MVP.

## Run

```bash
npm install
export NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
export NEXT_PUBLIC_ADMIN_TOKEN=dev-admin-token
npm run dev
```

## Routes
- `/` public home
- `/admin` admin entry
- `/admin/sources`
- `/admin/narratives`
- `/admin/claims/new`
- `/admin/evidence/new`
- `/admin/counterpoints/new`
- `/admin/action-signals/new`
